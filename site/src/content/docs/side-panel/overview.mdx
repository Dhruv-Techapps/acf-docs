The Side Panel is the quickest way to run automations and record new steps without leaving the current page.

## How To Open

Use any of the following methods:

1. Right-click on a page and choose <kbd>▤ Side Panel / ☉ Recording</kbd>.
2. Open the extension action menu and launch the Side Panel.

Once opened, the panel stays attached to the browser side and updates as you navigate.

## Tabs Overview

The Side Panel has two tabs:

1. `Automations`
2. `Recording`

### Automations Tab

- Shows the saved automation list.
- **Matched** group: automations that match the current site URL.
- **Other** group: all other saved automations.
- Click any automation row in the list to run it on the active tab.
- Click **Edit Automation** at the end of a row to open that automation in the Configuration page for editing.

### Recording Tab

- Use `Start Recording` to begin capture.
- Continue using the page normally (click, type, select, etc.).
- Steps are saved immediately as they are performed.
- Use `Stop Recording` to end capture.

For detailed recording behavior, see [Recording]([[docsref:/side-panel/recording]]).

## Footer Links

The Side Panel footer includes quick links for common actions:

- `Docs`: opens the documentation site.
- `Chat`: opens the community chat.
- `Advance`: opens the Configuration/Advanced page.
- `Sponsor`: opens the sponsorship page.

All footer links open in a new tab so your current page and Side Panel stay in place.
