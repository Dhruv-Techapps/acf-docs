Select one or more automations and remove them in a single action.

## Steps

1. Use the checkboxes next to each automation to select the ones you want to remove.
2. Use **Select all** to select every automation at once.
3. Click **Delete selected** and confirm the deletion.

<Callout type="warning">Removal is permanent. Export your automations first if you want a backup.</Callout>
