| **Notification Type**        | **Description**                                 |
| ---------------------------- | ----------------------------------------------- |
| **On Step completion**       | Notifies when an individual step is completed.  |
| **On Loop completion**       | Notifies when a loop of steps is completed.     |
| **On Automation completion** | Notifies when an automation is completed.       |
| **On Error**                 | Notifies when an error occurs during execution. |

</BsTable>

## Notification Sound

Toggle whether notifications play a sound.

## Notification on Discord

Connect the extension to a Discord server to receive notifications there. When enabled:

- Notifications (e.g., step complete, loop complete, errors) are sent to your Discord account.
- You must be part of the same Discord server and allow direct messages from server members.

<Callout type="warning">
  ! Important Enable notifications while configuring to catch errors quickly (e.g., missing/wrong XPath). You can turn
  them off once your setup is stable.
</Callout>
