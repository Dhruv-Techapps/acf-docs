## Search Inside Embedded Pages

Also look for items inside embedded sections of the page.

When enabled, the extension searches inside `<iframe>` elements embedded in the page before falling back to the main document.

## Refresh Page on Error

Automatically reload the page if something goes wrong.

If the extension context is invalidated (for example, after the extension is updated while the page is open), the page reloads automatically to re-establish the connection.

Show status bar at bottom of the page.

<BsTable>

| **Position**     | **Description**                                   |
| ---------------- | ------------------------------------------------- |
| **Hide**         | Hides the status bar entirely.                    |
| **Top Left**     | Places the status bar in the top-left corner.     |
| **Top Right**    | Places the status bar in the top-right corner.    |
| **Bottom Left**  | Places the status bar in the bottom-left corner.  |
| **Bottom Right** | Places the status bar in the bottom-right corner. |

</BsTable>
